# Банковское приложение 

## Оприсание проекта
Проект разрабатывается в рамках обучения и функционал все время растёт.
На данном этапе модули позволяют следующее:
- маскировать номер банковской карты и банковского счета.
- форматировать дату из европейского формата YYYY.MM.DD в российский ДД.ММ.ГГГГ.
- Фильтровать списки словарей по статусам выполнено или отменено.
- Фильтровать списки словарей по дате (по умолчанию пильтруется в поряде убывания).

## Структура проекта
### Корень проекта содержит:
    - .env.example - Модуль пример по образу которого создается локально модуль .env для присвоения 
                     переменной APILAYER_KEY токен - ключа с сайта: 
                     Exchange Rates Data API: https://apilayer.com/exchangerates_data-api, необходимого при
                     обращении к нему за кусом валют при преобразовании сумм транзакций в иностранной
                     валюте, для укузания её в рублях.
    - main.py - Временно неиспользуемый модуль.
### Папка src содержит следующие модули:
    - masks.py - содержит функции для маскировки номерок банковских карт и банковских счетов.
    - widget.py - маскирует номер банковской карты и банковского счета. 
                  форматирует дату из европейского формата в российский.
    - processing.py - содержит функции фильтрующие списки словарей по статусам и датам.
    - generators.py - содержит генераторы:
            - filter_by_currency - Фильтр списка транзакций по типу валюты.
              На вход принимает список транзакций.
              На выход отдает список транзакций.
            - transaction_descriptions - Фильтр списка транзакций.
              На вход принимает список транзакций.
              На выход отдает список с описаниями транзакций.
            - card_number_generator - Простой генератор номеров кредитных карт.
              На вход принимает два числа от 1 до 16-ти знаков.
              На выход отдает список номеров в указанном диапазоне,
              в формате "ХХХХ ХХХХ ХХХХ ХХХХ".
    - decorators.py - Декоратор для применения в журналировании любых функций.
    - external_api.py - функция принимает на вход транзакцию и возвращает сумму транзакции (amount) в рублях,
                        тип данных — float. Если транзакция была в USD или EUR, происходит обращение к внешнему API
                        для получения текущего курса валют и конвертации суммы операции в рубли.
    - utils.py - Модуль содержит функции для чтения файлов разного типа:
            - converting_data_from_json_to_a_dict_list - Функцию принимает на вход путь до JSON-файла
              и возвращает список словарей с данными о финансовых транзакциях.
              Если файл пустой, содержит не список или не найден, функция возвращает пустой список.
            - converting_data_from_csv_to_dict_list - Функцию принимает на вход путь до csv-файла
              и возвращает список словарей с данными о финансовых транзакциях.
              Если файл пустой, содержит не список или не найден, функция возвращает пустой список.
            - converting_data_from_xlsx_to_dataframe - Функцию принимает на вход путь до excel-файла
              и возвращает список словарей с данными о финансовых транзакциях.
              Если файл пустой или путь не верный, функция возвращает пустой список.
    - logging_masks.py - Логгер для журналирования работы функций модуля masks.py.
    - logging_utils.py - Логгер для журналирования работы функций модуля utils.py.
### Папка tests содержит такие модули тестов для testpy как:
    - test_mask_account.py - Тестирует маскировщик номера счета.
    - test_mask_card.py - Тестирует маскировщик номера карты.
    - test_processing.py - Теструет модуль processing.py на правильность работы фильтров по статусу и по дате.
    - test_widget_get_date.py - Тестирует правильность работы форматера даты.
    - test_widget_mask.py - Тестирует правильность маскировки номера счета или банковской карты.
    - test_generators_card_number_generator.py - Тестирует правильность генерации номеров кредитноых карт 
      в заданном диапазоне.
    - test_generators_filter_by_currency.py - Тестирует правильность фильтрации списка словарей по типу валюты.
    - test_generators_transaction_descriptions.py - Тестирует правильность генерации списа описанй транзакций 
      из полученного списка транзакций.
    - test_api_return_amount.py - Тестирует правильность работы модуля обращения к удаленному API
                                  для получения текущего курса валют и конвертации суммы операции в рубли.
    - test_decor_log.py - Тесты для декоратора на правильность логирования в файл и в консоль.
    - test_utils_data_converting.py - Содержит тесты на функции модуля utis.py:
            - Тест стения json - файла.
            - Тест стения csv - файла. (не реализовано)
            - Тест стения excel - файла. (не реализовано)

'''
2. Установите зависимости
'''
pip install -r requirements.txt
'''
