# Банковское приложение

## Описание проекта
Банковское приложение — это программный комплекс для обработки и анализа финансовых транзакций. Проект разрабатывался в рамках обучения и включает в себя широкий спектр функций для работы с банковскими данными.

Основной функционал приложения включает:
- Маскирование номеров банковских карт и счетов
- Конвертацию и форматирование дат
- Фильтрацию и сортировку транзакций по различным критериям
- Генерацию отчетов и аналитику финансовых операций
- Интеграцию с внешними API для получения курсов валют и акций
- Сервисные функции для финансового планирования

## Установка

1. Клонируйте репозиторий
```
git clone https://github.com/alekseyzvyagintsev/banking_app.git
```
2. Установите зависимости
```
poetry install
```

## Использование

Проект предоставляет два основных способа взаимодействия:

### 1. Через главный модуль main.py

Запустите приложение с помощью команды:
```
python main.py
```

Приложение предложит интерактивное меню для работы с банковскими транзакциями:
1. Выбор источника данных (JSON, CSV или XLSX файл)
2. Фильтрация операций по статусу (EXECUTED, CANCELED, PENDING)
3. Сортировка операций по дате (по возрастанию или убыванию)
4. Фильтрация только рублевых транзакций
5. Поиск операций по ключевому слову в описании

### 2. Через импорт модулей

Функционал приложения также доступен через прямой вызов функций из модулей. Основные возможности можно использовать, импортируя нужные модули и вызывая их функции с необходимыми аргументами.
